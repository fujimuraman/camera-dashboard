📈 Keepa API（全体販売数推定） オプション

Keepa の Sales Rank Drops から、各 ASIN の 30/90/180日間の推定販売数を取得し、 在庫一覧の「販売数 自分/全体」の全体側に表示します。 不要なら未入力でOK（販売数の自分側だけ表示されます）。

必要なもの: Keepa の有料 API key（https://keepa.com/#!api）
消費トークン: 1 ASIN あたり 1〜2 トークン。約 90 SKU を 24時間ごとに更新する想定なら月 2,700〜5,400 トークン消費。

保存後、定期 Polling（10分ごと）でも 24時間に1回まで自動同期します。

📊 カメラ市況分析（市場全体スコア） オプション

Amazon カメラ＆レンズの売れ筋トップ ASIN を継続取得し、市場全体の活況スコアを算出します。 売上分析グラフに「自社在庫スコア（紫）」と並べて「カメラ市場スコア（オレンジ）」を表示し、 市場全体が不振なのか自社固有の問題なのかを切り分けられます。

⚠️ 他に Keepa を使うツールがある場合は合算した使用量に注意してください。
100 ASIN を 24時間で均等取得（約14.4分に1件）。瞬間的にトークンを使い切らない設計です。
トークン残量が 50 を割ったら自動停止し、200 まで回復するまで待機します。
